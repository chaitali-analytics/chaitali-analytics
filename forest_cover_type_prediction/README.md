# Forest Cover Type Prediction using Machine Learning

## Project Overview

Forest Cover Type Prediction is a multiclass classification project that aims to identify the dominant forest cover type based on cartographic and environmental variables.

The project utilizes geographical and wilderness-related attributes such as elevation, slope, distance to hydrology, hillshade measurements, and fire point distances to predict the correct forest cover category.

Multiple machine learning models were trained and compared to evaluate their effectiveness in multiclass forest classification.

---

## Dataset

The dataset contains cartographic variables describing forest regions and their corresponding forest cover types.

### Features

- Elevation
- Aspect
- Slope
- Horizontal Distance to Hydrology
- Vertical Distance to Hydrology
- Horizontal Distance to Roadways
- Hillshade (9am)
- Hillshade (Noon)
- Hillshade (3pm)
- Horizontal Distance to Fire Points
- Wilderness Area Indicators
- Soil Type Indicators

### Target Variable

**Cover_Type**

Forest cover classes:

1. Spruce/Fir
2. Lodgepole Pine
3. Ponderosa Pine
4. Cottonwood/Willow
5. Aspen
6. Douglas-fir
7. Krummholz

## Outlier Handling

Outliers were detected and capped using the **Interquartile Range (IQR) Method**.

### Procedure

For each numerical feature:

- Calculate Q1
- Calculate Q3
- Compute IQR
- Define lower and upper bounds

```python
Lower Bound = Q1 - 1.5 × IQR
Upper Bound = Q3 + 1.5 × IQR
```

Values outside the bounds were capped rather than removed to preserve data integrity.
